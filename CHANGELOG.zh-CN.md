# 更新日志

本文档用于记录每个版本的主要变更，GitHub Actions 会自动提取对应版本内容生成 Release 说明。

## [0.1.1] - 2026-04-04

### 新增

- 新增一键发布脚本，可串联版本校验、本地打包检查、自动提交、打 tag 与推送远端。

### 修复

- 修复 Windows 发布资产收集逻辑，避免把体积很小的解包辅助 exe 错传到 GitHub Releases。
- 为 Windows 安装包增加体积校验，后续若再次收集到异常安装包会在 CI 中直接失败。

## [0.1.0] - 2026-04-03

### 新增

- 发布网站分身管理器桌面端首个版本。
- 支持站点管理，覆盖 `domain`、`entry_url`、`group` 三种模式。
- 支持分身管理，并为每个分身提供独立浏览器存储隔离。
- 支持内嵌浏览工作台与外置窗口两种打开方式。
- 支持悬浮式站点与分身管理抽屉、设置抽屉。
- 支持基于 GitHub Release 的更新检查与在线下载安装包。
- 支持多平台 GitHub Actions 自动打包发布流程。

### 说明

- macOS 发布资产包含 `aarch64`、`x64`、`universal` 三种 DMG。
- Linux 发布资产包含 `AppImage`、`deb`、`rpm`。
- Windows 发布资产包含 NSIS `.exe` 安装包。
